# Using The 文笔执行经验包

The `文笔执行经验包` is not a style imitation sheet. It is a living set of writing behaviors learned from author edits.

## Read It As Behavior

Apply entries as decisions such as:

- When to show instead of explain.
- How close the narrator should stand to emotion.
- Which kinds of dialogue sound false.
- Which sentence rhythms fit this book.
- Which scene entry patterns have worked.
- Which imagery families should be continued or avoided.
- Which AI habits the author repeatedly removes.

## Do Not

- Copy example wording from the pack.
- Mention the pack in prose.
- Apply a lesson outside its stated scope.
- Turn a flexible preference into an absolute rule.
- Let prose experience override fixed anchors, canon, or knowledge boundaries.

## Experience Strength

Treat each lesson by confidence:

| Confidence | How To Use |
| --- | --- |
| Candidate | Try lightly when it fits the chapter |
| Stable | Default to it unless the blueprint demands otherwise |
| Strong | Treat as a standing prose rule for its scope |
| Retired | Do not apply; preserved only for history |

## Conflict Rule

If a prose lesson conflicts with the chapter blueprint:

1. Obey fixed anchors and canon.
2. Preserve knowledge boundaries and character logic.
3. Use the prose lesson only inside the remaining expressive freedom.
