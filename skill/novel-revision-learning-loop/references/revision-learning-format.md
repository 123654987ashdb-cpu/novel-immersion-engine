# Revision Learning Format

## 作者改稿复盘

```markdown
# 作者改稿复盘 - 第 X 章

## 输入来源

| 来源 | 文件/位置 | 状态 |
| --- | --- | --- |
| 章节蓝图 |  |  |
| AI 初稿 |  |  |
| 作者修改稿/最终稿 |  |  |
| 已接受正文 |  |  |

## 蓝图目标

- 本章固定锚点：
- 本章边界内创造：
- 本章开放创造位：
- 本章禁止越界：

## 改稿对照

| 类型 | AI 初稿表现 | 作者修改 | 可能原因 | 影响 |
| --- | --- | --- | --- | --- |
| 结构/章节功能 |  |  |  |  |
| 人物/关系 |  |  |  |  |
| 场景执行 |  |  |  |  |
| 对话 |  |  |  |  |
| 情绪/内心 |  |  |  |  |
| 意象/氛围 |  |  |  |  |
| 句子/节奏 |  |  |  |  |
| 删除/压缩 |  |  |  |  |

## 根因判断

- 蓝图不够清楚：
- 蓝图写得过死：
- 正文端执行偏差：
- 作者局部审美选择：
- 值得吸收的新发现：
- 不应升级为规则的特例：

## 输出

- 总体经验增量：
- 文笔经验增量：
- 需要作者确认：
```

## 总体经验增量

```markdown
| 编号 | 经验 | 来源章节 | 适用范围 | 置信度 | 给策划端的动作 |
| --- | --- | --- | --- | --- | --- |
```

## 文笔经验增量

```markdown
| 编号 | 经验 | 作者修改证据 | 适用范围 | 置信度 | 给正文端的动作 |
| --- | --- | --- | --- | --- | --- |
```

## Scope Labels

- `本章特例`
- `当前阶段`
- `当前人物`
- `当前关系线`
- `当前题材/类型`
- `全书长期`
- `可迁移方法`

## Confidence Labels

- `候选`: one instance, plausible but untested.
- `稳定`: repeated or strongly supported.
- `强规则`: repeated and costly when violated.
- `退役`: once useful, no longer applicable.
